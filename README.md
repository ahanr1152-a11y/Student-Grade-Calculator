# 🎓 Student Grade Calculator (Java)

A simple console-based Student Grade Calculator developed in Java. This project calculates the total marks, percentage, grade, and pass/fail status based on the marks entered by the user.

## 📌 Features

- Enter student details
- Input marks for multiple subjects
- Calculate total marks
- Calculate percentage
- Display grade based on percentage
- Display pass/fail result
- User-friendly console interface

## 🛠️ Technologies Used

- Java
- Scanner Class
- Variables
- Arrays (if used)
- Loops
- Conditional Statements
- Methods

## 📂 Project Structure

```
StudentGradeCalculator.java
```

## 🚀 How to Run

1. Clone this repository.
2. Open the project in your preferred Java IDE (IntelliJ IDEA, Eclipse, or VS Code).
3. Compile the program:
   ```
   javac StudentGradeCalculator.java
   ```
4. Run the program:
   ```
   java StudentGradeCalculator
   ```

## 📊 Grade Criteria (Example)

| Percentage | Grade |
|------------|-------|
| 90% and above | A+ |
| 80% - 89% | A |
| 70% - 79% | B |
| 60% - 69% | C |
| 50% - 59% | D |
| Below 50% | Fail |

> *The grading criteria can be modified according to your requirements.*
